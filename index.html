O que é Big Data?

Big Data refere-se ao grande volume de dados - estruturados e não estruturados - que inundam as empresas diariamente. Mas não é a quantidade de dados que é importante. O que importa é o que as organizações fazem com os dados.

Volume

A quantidade de dados gerados

Velocidade

A velocidade de geração e processamento

Variedade

Os diferentes tipos de dados disponíveis
